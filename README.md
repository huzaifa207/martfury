# Bygg Och Fix

## To run client/front-end

* cd client
* yarn
* yarn dev

## To run server

* cd server
* yarn
* yarn dev


## For running the client and server on ngrok try following commands

move to the directory where ngrok file is located and run following command according to your OS.

* for linux
* ./ngrok http [port#]
* for windows
* ngrok.exe http [port#]
